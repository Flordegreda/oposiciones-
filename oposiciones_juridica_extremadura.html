Sobre esta aplicación

Aplicación de preparación de oposiciones para el cuerpo de Titulados Superiores de la Junta de Extremadura, especialidad jurídica.

El examen consta de 80 preguntas teóricas y 20 prácticas. Las preguntas erróneas penalizan 0,25 puntos. El temario cubre 75 temas de derecho público, administrativo, laboral e institucional.

Las preguntas se generan en tiempo real mediante la API de Claude (Anthropic). Los datos se guardan en tu navegador.

Zona de peligro

Elimina todos los datos guardados (estadísticas, bancos de preguntas y API key).
